Cols
含义:各列的信息；数量: 1


备注1: 上述列标题、buttonText，允许以"|"分隔中英文(用于中英文切换);
备注2: isHyperLink、isImage的列，如果加载的单元格数据以特征串 “[NoEffect]” 开头(大小写敏感)，表示该单元格以普通文字显示，特征串 [NoEffect] 使用范围为:
　　1.仅用于初始数据;
　　2.对 SetCellData( ) 函数设置的值无效;
　　3.该特征串在加载时能被自动识别、摘除，不会出现在单元格中;


Group　　
含义:分组表头；数量:0 - n;
属性名称含义可用值默认值
title(或name)显示的文字串无
headerFontIndex(或fontIndex)采用的字体, 前面定义的<Font>的序号数字-1
textColor(或color)文字颜色颜色串#000000
align文字对齐left/center/rightcenter
id分组的id，仅用于加载采用该id代替列名的XML/JSON数据串(无)
备注: 1.Group可以包含Col，也可以包含子Group，也可以和Col并列，相当于是树的树杈；
　　 2.title允许以"|"分隔中英文(用于中英文切换);

DropLists

Table

TD
